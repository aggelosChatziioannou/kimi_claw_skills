# Morning Briefing

Your daily morning ritual - everything you need to start the day right.

## What It Does

Every morning at 8:00 AM, receive a personalized briefing with:
- 🌤️ Weather for Ιωάννινα
- 📰 News summary (Ελλάδα, Tech, World)
- 🎯 Top 3 priorities for the day
- 💡 Morning motivation
- ⏰ Schedule preview

## Example Briefing

```
🌅 Καλημέρα, Boss!

📅 Παρασκευή, 13 Μαρτίου 2026

🌤️ Weather in Ιωάννινα:
   12°C • High: 16°/Low: 8° • Partly cloudy

📰 Today's Briefing:
   🏛️ Ελλάδα: New tech initiatives announced...
   💻 Tech: AI models show continued improvement...
   🌍 World: Major summit concludes with...

🎯 Today's Focus:
   • Review Kimi-Claw skill repository
   • Check on coffee shop website status
   • Plan weekend activities

💡 Morning Motivation:
   "The best way to predict the future is to create it."

⏰ Coming up:
   • 10:00 AM - Memory consolidation

Have a productive day! 🚀
```

## Manual Use

Request anytime:
- "Morning briefing"
- "What's my day look like?"
- "Brief me"

## Schedule

- **Default:** 8:00 AM daily
- **Timezone:** Europe/Athens
- **Channel:** Telegram
