# Kimi Claw Skills

Συλλογή από specialized skills για τον Kimi Claw AI assistant - οργανωμένα, τεκμηριωμένα και έτοιμα για χρήση.

## 📚 Διαθέσιμα Skills

### 1. [full-stack-artisan](./full-stack-artisan/)
**Master full-stack developer persona**

Συνδυάζει beautiful frontend design, solid backend architecture και comprehensive testing. Περιλαμβάνει:
- Next.js 14+ (App Router)
- TypeScript + Tailwind CSS + shadcn/ui
- Node.js/Python backends
- Jest + Playwright testing
- CI/CD deployment

**Χρήση για:** Web applications, full-stack projects, production-ready code

---

### 2. [classic-sequential](./classic-sequential/)
**Sequential workflow execution mode**

Ένας developer, πολλαπλά roles, sequential execution με continuous work:
- 5 phases: Research → Design → Development → Testing → Deploy
- Hourly heartbeat updates
- Πάντα διαθέσιμος για interruptions
- Automatic role switching

**Χρήση για:** Focused projects με clear timeline, single-threaded execution

---

### 3. [discrete-messaging](./discrete-messaging/)
**Communication protocol για async messaging**

Σπάει το thinking σε discrete, non-replaceable messages:
- Ποτέ δεν αντικαθιστά message (πάντα καινούργιο)
- Separate messages για thinking, status, actions
- Optimized για Telegram/async communication

**Χρήση για:** Telegram bot, async chat interfaces

---

### 4. [memory-consolidation](./memory-consolidation/)
**Daily memory refresh και consolidation system**

Διασφαλίζει ότι δεν ξεχνάω σημαντικές πληροφορίες:
- Καθημερινή επανάληψη στις 10:00 AM
- Consolidation 4 τύπων μνήμης (semantic, episodic, procedural, preferences)
- Auto-report στο Telegram
- Archive παλιών memories

**Χρήση για:** Long-term memory maintenance, consistency, personalization

---

### 5. [github-pro-sync](./github-pro-sync/) ⭐ NEW
**Zero-failure Git push protocol**

Διασφαλίζει ότι τα commits φτάνουν πραγματικά στο GitHub:
- Hash verification (local vs remote)
- Pre-flight checks
- Δεν λέει "έτοιμο" αν δεν δει hash match

**Χρήση για:** Reliable Git operations, verified deployments

---

### 6. [know-thyself](./know-thyself/) ⭐ NEW
**Kimi-Claw architecture και limitations**

Ξέρω τι μπορώ και τι δεν μπορώ να κάνω:
- Cloud-only constraints (no HTTP server, no local access)
- Daily update checks
- Pre-flight verification πριν προτείνω οτιδήποτε

**Χρήση για:** Αποφυγή impossible proposals, realistic planning

---

### 7. [communication-protocol](./communication-protocol/) ⭐ NEW
**Unified messaging standard**

Status → Content → Attachments (πάντα σε αυτή τη σειρά):
- Status updates πρώτα (🤔 → ⚙️ → ✅)
- Main content μετά
- Attachments πάντα ξεχωριστό message

**Χρήση για:** Καθαρή επικοινωνία, κανένα system noise

---

### 8. [transparent-thinking](./transparent-thinking/) ⭐ NEW
**Real-time progress visibility**

Βλέπεις τι κάνω καθώς το κάνω:
- Status messages κάθε 5-10 δευτερόλεπτα
- Progress indicators για complex tasks
- Κανένα πολύπλοκο task χωρίς updates

**Χρήση για:** Long-running tasks, research, analysis

---

### 9. [workspace-config](./workspace-config/) ⭐ NEW
**Configuration templates και SOUL.md**

Standardized workspace setup:
- SOUL.md με Communication Protocol
- AGENTS.md templates
- USER.md structure

**Χρήση για:** Consistent workspace organization

---

## 🚀 Πώς να Χρησιμοποιήσεις

- `README.md` - Full documentation

```

"Activate [skill-name] mode"
"Build [project] using [skill-name]"
```
