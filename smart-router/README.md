# Smart Skill Router

The brain of the system - automatically activates the right skills.

## What It Does

🧠 **Analyzes** your messages  
🎯 **Detects** what you need  
⚡ **Activates** the right skill automatically  
📚 **Learns** from your patterns  

## Auto-Activation Rules

| When you... | Auto-activates |
|-------------|----------------|
| Send code | 🔍 Code Reviewer |
| Say "remember/idea" | 🧠 Second Brain |
| Use git commands | 🔐 GitHub Pro Sync |
| Ask for research | 🔍 Deep Research |
| Plan project | 📋 Task Planner |

## How It Works

```
You: "Here's my code..."
↓
Router: Analyzes content
↓  
Router: Detects code → 95% confidence
↓
Router: Auto-activates Code Reviewer
↓
You: Get code review instantly!
```

## Confidence Levels

- **>90%** → Silent activation
- **70-90%** → Activate with notification
- **<70%** → Ask for confirmation

## Learning

The router learns from your behavior:
- Tracks successful activations
- Adjusts confidence thresholds
- Adapts to your patterns
