# Second Brain

AI-powered personal knowledge management.

## What It Does

Captures, organizes, and synthesizes your knowledge:

📝 **Capture:** Brain dumps, meeting notes, ideas  
🗂️ **Organize:** Auto-categorization and tagging  
🔗 **Connect:** Links related ideas  
💡 **Synthesize:** AI insights and patterns  

## Folder Structure

```
second-brain/
├── inbox/      # Quick captures
├── daily/      # Daily notes
├── projects/   # Active projects
├── areas/      # Ongoing interests
├── resources/  # References
├── archives/   # Completed
└── insights/   # AI syntheses
```

## Usage

### Capture Idea
```
"Capture this: My idea about AI automation"
"Add to second brain: Meeting notes from today"
```

### Search Knowledge
```
"Search my second brain for automation"
"What do I know about web development?"
```

### Daily Insights
```
"Generate weekly insights"
"Second brain daily brief"
```
